GAME SET BLOG is personal blog cataloging analysis and opinion on ATP, WTA, and Grand Slam Tennis. The application is built using React and connected to a headless Wordpress backend. The site uses an Incremental Static Generation (ISG) rendering strategy to boost performance and SEO.
